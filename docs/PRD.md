# Product Requirements Document (PRD)

## Overview
A Single User Todo Application that supports creating basic todos and assigning them to categories.

## Architecture Context
- **Frontend**: React, Tailwind CSS.
- **Backend**: Python FastAPI.
- **Database**: Local SQL Server Express Edition (Windows Auth).

## Milestones
1. **Basic Todo Foundation**: CRUD operations for a Todo (no categories).
2. **Unit Testing**: 100% logic coverage with mocks.
3. **Category Integration**: Add Category creation and assignment.
4. **Edge Case Handling**: Enforce robustness and standard HTTP errors.
